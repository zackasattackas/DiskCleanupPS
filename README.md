# DiskCleanupPS
A binary PowerShell module containing cmdlets for performing various disk cleanup tasks.

## Exported Cmdlets

```PowerShell
Get-ChildFileSystemItem
Get-FileSystemItemSize
Get-UserProfile
Invoke-CCleaner
New-NtfsCompressionOptions
Remove-TempFiles
Set-NtfsCompression
```
